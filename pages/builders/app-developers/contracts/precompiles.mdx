# Precompiled Contracts

## Bitcoin State Oracle

GOAT Network provides a built-in Bitcoin state Oracle for the on-chain applications. This allows the developers to implement the [SPV](https://bitcoinwiki.org/wiki/simplified-payment-verification) directly via the method [blockHash](https://github.com/GOATNetwork/goat-contracts/blob/main/contracts/bitcoin/Bitcoin.sol#L36) in smart contract [Bitcoin](https://github.com/GOATNetwork/goat-contracts/blob/main/contracts/bitcoin/Bitcoin.sol) and [WrappedBitcoin]()https://github.com/GOATNetwork/goat-contracts/blob/main/contracts/bitcoin/WrappedBitcoin.sol on GOAT Network.
