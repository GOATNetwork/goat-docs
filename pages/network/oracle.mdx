# Basics

The GOAT Network Bridge enables secure asset transfers between Bitcoin (L1) and GOAT Network (L2) through ZKM's Trustless Bridge Protocol. This guide lists all the Oracles that dApps can use.

<Callout type="info">
  New to GOAT Network? Visit the [GOAT Testnet 3 Faucet](https://bridge.testnet3.goat.network/faucet) to get started with test BTC.
</Callout>
