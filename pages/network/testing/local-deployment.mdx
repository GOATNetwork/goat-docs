# Running a Local Development Environment
This tutorial teaches you how to run a local development environment of the entire GOAT Mainnet system.

## What is this?

A development environment is a local installation of the entire GOAT Mainnet system.
Our default development environment includes both L1 and L2 development nodes.
Running the GOAT Mainnet environment locally is a great way to test your code and see how your contracts will behave on GOAT Mainnet before you graduate to a testnet deployment (and eventually a mainnet deployment).
