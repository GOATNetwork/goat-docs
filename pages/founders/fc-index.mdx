1. **Community-Driven Growth**
   - Build a loyal and engaged user base
   - Receive direct feedback and support
   - Collaborate with like-minded innovators

2. **Advanced Analytics**
   - Track Daily Active Users (DAU)
   - Monitor Transaction Volumes (TRX)

3. **Flexible Participation**
   - Join at any project stage
   - Create custom engagement quests
   - Align user actions with project goals

## Participation Tracks

<Callout type="warning">
  Three dynamic tracks to suit your project's unique needs and growth strategy
</Callout>

1. **New Project Development**
   - Ideal for early-stage blockchain applications
   - Focused on initial user acquisition
   - Maximum points: 500,000

2. **Retroactive Rewards**
   - For existing projects seeking expansion
   - Based on performance metrics
   - Maximum points: 1,000,000

3. **User Growth Acceleration**
   - Emphasizes daily active user growth
   - Tiered rewards based on user adoption
   - Points vary by monthly user milestones
